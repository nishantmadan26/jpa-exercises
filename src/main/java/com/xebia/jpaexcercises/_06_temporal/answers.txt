After executing the DbApp the @Temporal annnotation used to insert the date format data in the DB.
1) TemporalType.DATE is saved the dat in default YYYY-MM-DD format while
2) TemporalType.TIMESTAMP is used to saved the date as well as the timestamp of that time in default format
    YYYY-MM-DD HH:MM:SS.But we can change the format of the date as per our convinence.
